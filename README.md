# cooking-quiz-for-eng-class
this was a short quiz made in html for a specific food related project for english class, it utilizes base64 embedding via a nodejs script to render audio clips and maximize security and portability across devices since html files dont have the ability to use functions to call for files natively because of heavy sandboxing done by mainstream browsers
this effectively removes the need for custom electron applications that run very ineffeciently and may or may not be detrimental to security 
it's also worth mentioning how this allowed me to compile comfortably in my linux machine instead of using wine or a windows vm to compile individual binaries for every os
